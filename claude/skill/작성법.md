| 속성                           | 기능                                      |
| ------------------------------ | ----------------------------------------- |
| disable-model-invocation: true | 사용자만 호출 가능, Claude 자동 호출 불가 |
| user-invocable: false          | Claude만 호출 가능, / 메뉴에서 숨김       |

`disable-model-invocation`은 A 스킬이 false, B 스킬이 true라고 한다면 A스킬이 사용자에 의해 호출되면 해당 세션에서는 클로드가 B스킬도 호출 가능하다.
