## 쓰고 있는 값이 ==어떤 자료형==인지 알 수 있음

### int (정수형)
```python
print(type(3))

=> <class 'int'>
```

### float (소수형)
```python
print(type(3.0))

=> <class 'float'>
```

### str (문자형)
```python
print(type("3"))

=> <class 'str'>
```

### bool (불린)
```python
print(type(True))

=> <class 'bool'>
```

### function (함수)
```python
def hello():
    print("Hello world!")

print(type("3"))

=> <class 'function'>
```
