- **자손 결합자 (공백):** `div p`
    - `div` 안에 있는 **모든** `p`를 선택합니다. (아들, 손자, 증손자 다 포함)
- **자식 결합자 (`>`):** `div > p`
    - `div`의 **바로 아래 단계(직계 자식)**인 `p`만 선택합니다. (손자는 제외)
    - 구조가 복잡해질수록 `>`를 써서 범위를 좁히는 것이 안전합니다.
- **인접 형제 결합자 (`+`):** `h1 + p`
    - `h1` **바로 다음에 나오는** `p` 하나만 선택합니다.
- **일반 형제 결합자 (`~`):** `h1 ~ p`
    - `h1` 뒤에 나오는 형제들 중 **모든** `p`를 선택합니다.

### nth 프리픽스

- **`:nth-child`**: **"모든 형제들 중에서"** 몇 번째인지 셉니다. (태그 종류 상관 안 함, 일단 줄부터 세움)
- **`:nth-last-child`**: `:nth-child`와 같지만 뒤에서부터 검사한다.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        p:nth-child(2){ /* 2번째 자식인데 p인 요소 */
            height: 20px;
            background-color: red;
            width: 20px;
        }
    </style>
</head>
<body>
    <div>
        <p>1</p>
        <span>2</span> <!-- 2번째 자식이지만 p가 아님 -->
        <p>3</p> <!-- 2번째 p이지만 2번째 자식이 아니라서 해당 안 됨 -->
    </div>
    <div>
        <p>4</p>
        <p>5</p> <!-- RED! 2번째 자식이고 p라서 red -->
        <p>6</p>
    </div>
</body>
</html>
```

    
- **`:nth-of-type`**: **"같은 태그들 중에서"** 몇 번째인지 셉니다. (다른 태그는 투명인간 취급)
- **`:nth-last-of-type`**: 위와 마찬가지로 `nth-of-type`을 뒤부터 검사

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        p:nth-of-type(2){ /* p중에 2번째로 나타나는 자식 */
            height: 20px;
            background-color: red;
            width: 20px;
        }
    </style>
</head>
<body>
    <div>
        <p>1</p>
        <span>2</span> <!-- 2번째 자식이지만 p가 아니라서 노카운트! -->
        <p>3</p> <!-- RED! 3번째 자식이지만 2번째로 나타난 p요소! -->
    </div>
    <div>
        <p>4</p>
        <p>5</p> <!-- RED! 2번째 자식이고 p! -->
        <p>6</p>
    </div>
</body>
</html>
```
