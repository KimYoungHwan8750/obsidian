# Needle: We Distilled Gemini Tool Calling Into a 26M Model

## 요약

`Needle`이라는 `26M` 파라미터 function-calling 모델을 오픈소스로 공개했습니다.
`Gemini`의 tool calling을 distilled 했고, consumer device에서 높은 `tok/s` 성능을 보인다고 합니다.
에이전틱 경험의 핵심을 reasoning보다 tool selection과 argument extraction으로 본 점이 특징입니다.
작은 모델로 함수 호출을 최적화한 아키텍처 제안이라는 점에서 주목할 만합니다.
