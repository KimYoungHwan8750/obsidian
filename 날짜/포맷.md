| **구분**         | **PHP date()** | **SQL(MySQL)** | **일반 명칭(참고)** |
| ---------------- | -------------- | -------------- | ------------------- |
| **연도 (4자리)** | `Y`            | `%Y`           | `YYYY`              |
| **월 (01-12)**   | `m`            | `%m`           | `MM`                |
| **일 (01-31)**   | `d`            | `%d`           | `DD`                |
| **시 (00-23)**   | `H`            | `%H`           | `HH24`              |
| **분 (00-59)**   | `i` (중요!)    | `%i`           | `MI` / `mm`         |
| **초 (00-59)**   | `s`            | `%s`           | `SS`                |

일반 명칭은 Java, Moment.js에서 사용되는 형식이다.
