```java
String aiContents = params.getString("data["+i+"][aiContents]");
String aiCode = params.getString("data["+i+"][aiCode]");
```

파라미터로 리스트를 전달하게 되면 인덱스와 키가 하나의 문자열로 오게 된다.

```js
const data = {
	aiContents: "data1",
	aiCode: "data2"
}
```

따라서 위와 같은 데이터 형식에서 aiCode를 가지고 오고 싶다면 `"data[0][aiCode]"`자체를 키로 사용해야 불러올 수 있다.

CommonMap의 메서드인 getList를 사용해 반환 받은 CommonMap을 사용해야 `.getString("aiCode")`와 같이 사용 가능하다고 보여지는데, 이는 직접 해보고 이 노트의 내용을 갱신해야할 필요가 있다.
