### 오픈소스 패키지

| 도구                            | 특징                                          |
| ----------------------------- | ------------------------------------------- |
| **OCRmyPDF**                  | 스캔 PDF에 텍스트 레이어 추가, Tesseract 기반, CLI 사용 가능 |

| **EasyOCR**                   | 80개+ 언어 지원, 딥러닝 기반, 설치 간단                   |
| **PaddleOCR**                 | 한국어 포함 다국어, 표/레이아웃 인식 강함                    |

**MiniCPM-o 2.6** — 8B 파라미터로 OCRBench 1위, GPT-4o보다 높은 점수, 30개 언어 지원, 모바일 배포 가능. **Mistral OCR** — PDF/표/수식 처리에 강하고 마크다운으로 출력, 1,000페이지당 $1. **Qwen2-VL** — Alibaba의 오픈소스 비전-언어 모델.

## 🔗 2단계: OCR 텍스트 → RAG 파이프라인 (핵심 연결 구조)

```
ebook/PDF → OCR → 텍스트 청크 → 임베딩 → 벡터 DB → LLM 질의응답
```

### 주요 스택 조합

OCR로 텍스트 청크를 추출한 뒤, Word2Vec/BERT 같은 임베딩 모델로 벡터화하여 ChromaDB 같은 벡터 DB에 저장합니다. 사용자 질문도 같은 임베딩 모델로 벡터화해서 유사 청크를 검색한 뒤 LLM에 전달하는 구조입니다.

**인기 조합:**

- **LangChain + ChromaDB/FAISS + OpenAI/Ollama** — 가장 보편적
- **LlamaIndex + AstraDB** — 대규모 문서에 유리
- **FastAPI + FAISS + Mistral/DeepSeek** — 로컬 완전 오프라인 구성 가능

# 4. 튜터 LLM

# 5. UI
Streamlit (빠른 프로토타입)
