## 포트에 대한 방화벽 제어
```
# 8000번 포트 개방
firewall-cmd --permanent --add-port=8000/tcp
# 방화벽 규칙 적용
firewall-cmd --reload
```

```
# 8000번 포트 개방 규칙 삭제
firewall-cmd --permanent --remove-port=8000/tcp

# 설정 반영
firewall-cmd --reload
```

## 아이피에 대한 방화벽 제어
```
# Rich Rule을 사용하여 특정 IP(예: 192.168.0.62)에만 8000번 허용
firewall-cmd --permanent --add-rich-rule='rule family="ipv4" source address="192.168.0.62" port port="8000" protocol="tcp" accept'

# 특정 IP에 대한 Rich Rule 제거
firewall-cmd --permanent --remove-rich-rule='rule family="ipv4" source address="192.168.0.62" port port="8000" protocol="tcp" accept'

# 2. 설정 즉시 반영
firewall-cmd --reload
```
