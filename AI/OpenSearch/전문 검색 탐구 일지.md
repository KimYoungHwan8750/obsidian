## Nori를 왜 쓸까?

우선 "김치찌개가"라는 키워드로 쿼리를 날린다.

```json
GET /recipe/_search
{
  "query": {
    "match": {
      "content": "김치찌개가"
    }
  }
}
```

```json
{
  "took": 5,
  "timed_out": false,
  "_shards": {
    "total": 1,
    "successful": 1,
    "skipped": 0,
    "failed": 0
  },
  "hits": {
    "total": {
      "value": 0,

    },
```

검색 결과는 없다고 뜬다.

nori를 사용하지 않으면 형태소를 구분하지 못해 김치찌개가, 김치찌개는, 김치찌개랑 같은 단어들이 모두 별개의 키워드가 된다.

하지만 nori는 조사 은, 는, 이, 가 등 심층적인 연구결과에 기반하여 키워드에 크게 의미없는 형태소를 제거하고 토큰화해준다.

이제 analyzer를 nori로 설정한 후의 결과를 보자.
