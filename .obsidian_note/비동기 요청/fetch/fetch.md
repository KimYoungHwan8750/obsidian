## fetch()

```javascript
fetch("url",{options}).then().catch();
```

#### options에는
```javascript
method:"post" // "post","put"등이 들어갈 수 있다.
headers:{} // headers 안에는 "Content-Type":"application/json" 등 요청-응답 포맷을 설정할 수 있다.
body: // 원하는 데이터를 실어보내면 된다.
```
