# 개요

효율적인 공부 체계를 제공하여 쉽게 영단어를 습득하게 도와주는 어플리케이션

## 개발 환경
	1. React Native (Framework)
	2. Firebase (DB)
	3. Redux (상태관리)
	4. react-navigation (네비게이션)
	5. AsyncStorage (로컬저장소)
	6. OpenAI (AI 활용 컨텐츠)
	7. react-native-vector-icon (icon)
