#### 사용법
slice(시작 인덱스, 종료 인덱스)
```js
let test = [0,1,2,3,4,5];
test.slice(0,2); // [0,1]
test.slice(2,4); // [2,3]
test.slice(0,-2); // [0,1,2,3]
```

시작 인덱스는 0부터 시작한다.
범위에 음수를 넣으면 뒤에서부터 인덱스 검사를 시작한다.
만약 범위에 -3을 넣으면 뒤에서 3번째 숫자의 앞까지 자른다.
[[splice]]와 차이점은 slice는 원본에 영향을 주지 않고 결과값만을 반환하지만 splice는 결과값을 원본 배열에 반환한다.
