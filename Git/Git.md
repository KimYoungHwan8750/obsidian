# Git
버전 관리용 툴

## Git 명령어

| 명령어                     | 동작                                      |
| -------------------------- | ----------------------------------------- |
| git status                 | 현재 상태 확인                            |
| git add                    | 스테이징 영역에 파일 추가                 |
| git commit -m "msg"        | 스테이징 영역 파일들 메세지와 함께 커밋   |
| git push                   | 커밋 적용                                 |
| git remote add 이름 "주소" | 이름으로 해당하는 주소 원격 저장소로 등록 |
|                            |                                           |
