## 중앙 저장소 직접 만들기

--bare 옵션을 통해 해당 프로젝트 정보가 담긴 .git 파일을 생성할 수 있다.

### 처음부터 중앙 저장소로 사용

```bash
git init --bare
```

--bare 옵션을 주게 되면 소스코드 내용 없이 이력 관리만 되는 git 폴더가 생성된다.


### 기존 프로젝트를 중앙 저장소로 마이그레이션

```bash
git init
git add .
git commit -m "init"
git clone --bare . <목표경로>
```

예를 들어 `git clone --bare . C:/path`를 입력하면 C:/path에 .git 폴더가 생성된다. clone 하기 전에 현재 소스 코드 내용을 담으려면 꼭 add 와 commit을 수행해줘야한다.

이렇게 생성된 C:/path를 A, B, C 개발자가 사용하려면 각각 git clone C:/path를 입력하면 된다.
