리포지토리 폴더에 `.github/workflows` 폴더를 만든다.
workflows 폴더 내에 있는 yml 파일에 정의된 내용이 실행된다.

```yaml
steps:
  - uses: actions/checkout@v4        # 1. 코드 가져오기
  - uses: actions/setup-node@v4     # 2. 노드 설치
    with:
      node-version: '20'
  - uses: actions/cache@v4           # 3. 캐시 적용 (빠른 빌드)
    with:
      path: ~/.npm
      key: ${{ runner.os }}-node-${{ hashFiles('**/package-lock.json') }}
  - run: npm install
  - run: npm run build
  - uses: actions/upload-artifact@v4 # 4. 빌드 결과물 저장
    with:
      name: my-dist
      path: dist/
```

샘플 코드는 위와 같다.
